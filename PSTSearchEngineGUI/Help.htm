 Setup

Run the setup file and follow the instructions

Click Next

Select Destination Folder and click next

Confirm installation and click next

Click close when the installation is complete.

 

Running the program

When the program is first launched it will ask for a catalogue folder.

Click Yes

Point to the required folder for instance click Computer à Local Disk (C:) à OSE_Catalogue (you can create folders using make new folder)

After selecting the folder click OK.

The program will run in trial mode where it can retrieve up to 5 emails per search

Get License

Go to help registration

In the registration box that pops up write a name in the edit box and click get license

When the license file is received it should placed in the installation folder of the application (for instance “C:\Program Files\CK Development\PST Search Engine”)

Define Catalogue

Got to file à change catalogue path

Select the path and press OK

Build Catalogue

There are two options to build catalogue:

1-      Index one PST at a time.

2-      Index all the PST that are loaded to outlook.

It should be noted that to run build catalogue command the outlook must be open and the target PST shall be loaded to outlook.

Index one PST at a time go to file àIndex Single PST

The outlook folder pick dialog becomes visible

Select the a folder within the PST you want to index and press OK

The indexing starts in a separate thread. When completed an OK dialog box will pop up

Click ok.

 

Search

In order to know how many email are indexed and what are the possible searchable header click find button while keeping the textbox empty

In order to find an email, type the filter in the available text box and press find respecting the following rules:

The right hand side shall be the column heading separated by a “:” character from the filter variable. There should not be any space between the “:” and the filter variable

[Column Heading]:[Filter Variable]

“*” character acts a wild card

Multi filters can be separated by “;” character.

The list of column heading is as follows:

Column Header

Meaning

From

 The sender property of the email

To

 The To property of the email

CC

 The CC property of the email

ToCC

 All the recipient of the email

MailDate

 Mail creation date

MailTime

 Mail creation time

Subject

 The email subject

Body

 The email body text

AttachmentsName

 The file Name of the attachments

AttachmentsCount

 The number of attachments

FullFolderPath

 The location of the email within the pst

 

Search examples:

From:ch*;to:zu*   à this filter gives all the email originating from a sender whose name starts with ch to a recipient whose name starts with zu

From:*john*;body:*new jersy*  à this filter gives all the email originating from a sender whose name contains with john and the mail text contains new jersy
